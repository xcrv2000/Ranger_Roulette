# 漫游枪手卡包接入计划（2026-04-17）

## Summary
- 将 `设计方案/漫游枪手子弹包.md` 中 37 张卡接入项目：更新 `data/cards.json`、新增/替换对应 `scripts/cards/*.gd` 脚本。
- 规范化卡牌文案为“关键词风”：保留关键词（如【消耗】/【补射】），统一标点与换行，并将所有“打X”改为“造成X点伤害。”。
- 不改动战斗系统结算逻辑（保持当前仅 `damage/block/move` 生效）；复杂效果先在脚本结构中保留占位信息，保证可加载。
- 增加卡牌 `random_pool` 字段：设计卡为 `漫游枪手`，`test1/test2/test3` 为 `测试`；在奖励三选一中仅启用 `漫游枪手` 池，使测试卡不再出现。
- “假目标”卡仅记录状态/占位，不做 UI 立绘切换。

## Current State Analysis
- `data/cards.json` 当前仅 5 张卡：`attack/defend/test1/test2/test3`，字段包含 `id/name/text/icon/anim_module/script`。
- `scripts/cards/` 当前仅有 5 个脚本，`get_effect()` 返回 `{damage, block, move}`。
- `scripts/cards/card_script_resolver.gd` 仅调用脚本 `get_effect()` 与可选 `get_timeline_seconds()`。
- `scripts/battle/battle_controller.gd` 结算时只读取 `damage/block/move` 三字段，未支持“消耗/补射/随机打出/延迟触发/立绘变化”等机制。
- `scripts/reward/reward_controller.gd` 三选一卡池来自 `CardDatabase.get_all_ids()`，当前未按池过滤。

## Proposed Changes

### 1) 更新卡牌数据：`data/cards.json`
- 用 37 张“漫游枪手”卡替换现有设计入口数据，并保留 `test1/test2/test3` 用于测试隔离。
- 每张卡新增字段：`random_pool`。
- `random_pool` 规则：
  - 37 张新卡：`"random_pool": "漫游枪手"`
  - `test1/test2/test3`：`"random_pool": "测试"`
  - 现有 `attack/defend`：为避免污染奖励池，也标记为 `测试`（或移除，最终以实现一致性为准）。
- `text` 字段执行文案正规化：
  - 统一全角标点。
  - 以关键词段落表达（如“【消耗】……”“【补射】……”）。
  - 全量替换“打X”为“造成X点伤害。”。
  - 拆分长句为可读换行（`\n`）。

### 2) 生成卡脚本：`scripts/cards/*.gd`
- 为 37 张卡逐张创建脚本文件（命名与 `cards.json` 的 `id` 对齐）。
- 脚本接口保持兼容：
  - `static func get_effect() -> Dictionary`
  - `static func get_timeline_seconds() -> float`
- 因不扩展战斗系统，本次策略：
  - 可直接映射的效果写入 `damage/block/move` 基础值（如“前进/后退”“造成X点伤害”“获得X格挡”）。
  - 复杂机制以扩展字段保留（如 `tags`, `notes`, `design_flags`），并确保不影响现有结算。
  - “假目标”仅写占位标记（例如 `design_flags.fake_target = true`），不做立绘资源调用。

### 3) 随机池接入：`scripts/reward/reward_controller.gd`
- 在 `_roll_card_options()` 中按 `random_pool` 过滤候选卡。
- 新增活动池常量（或局部配置）`ACTIVE_RANDOM_POOL = "漫游枪手"`。
- 只从 `random_pool == ACTIVE_RANDOM_POOL` 的卡中抽取三选一。
- 当池内数量不足时，按“可选数量上限”安全返回，避免空引用。

### 4) 稳定性兼容调整（最小范围）
- 若 `run_context` 初始轮盘仍引用被移除卡 id，则同步把初始轮盘改为存在的卡 id（仅数据层调整，不改战斗机制），避免出现“未知技能”刷屏。
- 若保留 `attack/defend`，则确保它们不进入 `漫游枪手` 奖励池。
